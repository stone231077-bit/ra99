# ra99.in.io - Сайт-визитка

Сайт-визитка для магазина електротоварів ra99.in.io

## Структура проекта

- `index.html` - основная страница сайта
- `styles.css` - стили оформления
- `script.js` - JavaScript для формы обратной связи

## Функционал

- ✅ Информация о команде с контактами
- ✅ Каталог брендов (Schneider, ABB, Konlux, Horos, Firon, Novodvorski)
- ✅ Информация о товарах (розетки, выключатели, светильники, провода, автоматика)
- ✅ Адрес магазина
- ✅ Форма обратной связи с выбором типа запроса (заказ, предложение, отзыв, жалоба)

## Запуск

Просто откройте файл `index.html` в браузере.

## Настройка отправки формы

Текущая версия использует `mailto:` для отправки писем. Для полноценной работы через сервер:

### Вариант 1: Formspree (рекомендуется)
1. Зарегистрируйтесь на https://formspree.io
2. Получите ваш Form ID
3. Замените код в `script.js` на отправку через Formspree API

### Вариант 2: EmailJS
1. Зарегистрируйтесь на https://www.emailjs.com
2. Настройте сервис и обновите код в `script.js`

### Вариант 3: Backend
Настройте свой сервер для отправки почты и обновите URL в `script.js`

## Добавление фотографий

Когда будут готовы фотографии сотрудников, замените блоки `.photo-placeholder` в `index.html` на:

```html
<img src="path/to/photo.jpg" alt="Имя сотрудника" class="team-photo">
```

И добавьте в `styles.css`:

```css
.team-photo {
    width: 200px;
    height: 200px;
    object-fit: cover;
    border-radius: 15px;
    margin: 0 auto 1.5rem;
    display: block;
}
```
