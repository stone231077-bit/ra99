# Сайт услуг - Адаптация дизайна Miks Soft

Этот проект представляет собой адаптацию дизайна сайта Miks Soft под ваши нужды.

## Структура проекта

- `index.html` - основная HTML страница
- `styles.css` - стили CSS для современного дизайна
- `script.js` - JavaScript для интерактивности

## Особенности

- ✅ Современный адаптивный дизайн
- ✅ Мобильное меню
- ✅ Плавная прокрутка
- ✅ Анимации при прокрутке
- ✅ Форма обратной связи с валидацией
- ✅ Cookie баннер
- ✅ Адаптивная верстка для всех устройств

## Как использовать

1. Откройте `index.html` в браузере
2. Или используйте локальный сервер:
   ```bash
   # Python
   python -m http.server 8000
   
   # Node.js
   npx http-server
   ```

## Настройка под себя

### Изменение контента

1. **Название компании**: Замените "Ваша Компания" на ваше название в `index.html`
2. **Контакты**: Обновите контактную информацию в футере
3. **Услуги**: Измените карточки услуг в секции `.services`
4. **Технологии**: Обновите список технологий в секции `.technologies`
5. **Цвета**: Измените CSS переменные в `styles.css`:
   ```css
   :root {
       --primary-color: #000;
       --accent-color: #007bff;
       /* и т.д. */
   }
   ```

### Интеграция формы

Для отправки формы на сервер, замените обработчик в `script.js`:

```javascript
contactForm.addEventListener('submit', async (e) => {
    e.preventDefault();
    const formData = new FormData(contactForm);
    
    try {
        const response = await fetch('/api/contact', {
            method: 'POST',
            body: formData
        });
        // Обработка ответа
    } catch (error) {
        console.error('Error:', error);
    }
});
```
