# Быстрая настройка отправки формы через Formspree

## Шаг 1: Регистрация
1. Перейдите на https://formspree.io
2. Нажмите "Sign Up" (можно через Google/GitHub)
3. Это бесплатно для до 50 заявок в месяц

## Шаг 2: Создание формы
1. После регистрации нажмите "New Form"
2. Скопируйте ваш **Form ID** (например: `xrgkqjpn`)

## Шаг 3: Активация
1. Откройте файл `script.js`
2. Найдите строку:
   ```javascript
   const FORMSPREE_ID = null;
   ```
3. Замените `null` на ваш Form ID:
   ```javascript
   const FORMSPREE_ID = 'xrgkqjpn'; // Ваш Form ID
   ```

## Готово! ✅
Теперь форма будет отправлять заявки напрямую на почту `vasil.ra99@gmail.com`

## Альтернатива: EmailJS
Если хотите использовать EmailJS:
1. Зарегистрируйтесь на https://www.emailjs.com
2. Настройте сервис и шаблон
3. Раскомментируйте и заполните переменные в `script.js`
